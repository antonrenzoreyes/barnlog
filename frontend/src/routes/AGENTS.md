# Routes Agent Guide

## Scope

- This file applies to all work under `frontend/src/routes/`.

## Route UX Skills

- Use `$onboard` to improve first-time UX, empty states, and activation paths.
- Use `$adapt` to make route and page UX work across breakpoints and device contexts.
- Use `$clarify` to improve labels, instructions, and user-facing errors in page flows.
- Use `$delight` to add tasteful personality moments in key interactions and transitions.
- Use `$animate` to add purposeful motion for feedback and flow.
- Use `$optimize` to improve route-level performance and perceived speed.
- Use `$harden` to cover loading, error, empty, overflow, and i18n edge states.
- Use `$overdrive` for advanced interaction treatment only when explicitly requested.
